关于此页面

这个页面是一个静态 HTML 模板，用于测试 GitHub Pages 的部署功能。GitHub Pages 可以让你直接从 GitHub 仓库托管静态网站。

如果页面正常显示，说明你的 GitHub Pages 已成功运行。你可以随意修改此页面的代码，添加自己的项目、博客或作品集。

🔧 简单部署

只需将 HTML、CSS、JS 文件推送到指定分支（如 `main` 或 `gh-pages`），GitHub 会自动构建并发布你的网站。

🎨 完全自定义

你可以使用此模板作为起点，自由修改样式、添加组件或集成任何前端框架（如 Vue, React）。

🌐 免费托管

GitHub Pages 为每个账户提供一个免费的 username.github.io 域名，并支持绑定自定义域名。
